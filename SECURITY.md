# Security Policy

## Supported Versions
- This repository is provided "as-is". For high-impact issues, please report privately.

## Reporting a Vulnerability
Please report security or safety issues via:
- Email: <siriusa.paper@gmail.comL>
- Subject: [SECURITY] <short title>

## Scope
- Vulnerabilities, safety risks, or misuse vectors related to code/docs in this repo.

## What to include
- Steps to reproduce / Proof of concept (optional)
- Impact assessment (who/what can be harmed)
- Suggested mitigation (if any)

## Response
- Initial response: within 7 days (best-effort)
