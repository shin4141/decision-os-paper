# As-of Pack (v0)

As-of（当時点固定）を「内容固定」ではなく **評価の固定（裁き方の固定）**として運用するためのテンプレ群です。

## Download
- **PDF（印刷用）**: `ASOF_PACK_v0.pdf` *(coming soon)*
- **DOCX（編集用）**: `ASOF_PACK_v0.docx` *(coming soon)*
- **MD（コピペ用）**: `ASOF_PACK_v0.md`

## How to use (最短)
1. `ASOF_PACK_v0.md` を開く  
2. **全選択 → コピー → 自分のノート/新チャット先頭に貼る**  
3. 先頭でこれを宣言する：  
   - `Default=Coauthor / Critic-ON only / Forward-only`

## Rule (大事な前提)
- Defaultは **Coauthor**（過去を裁かない）
- Criticは **明示ONのみ**
- 変更は **Forward-only（差分追記）**：過去を否定せず、Addendumで増やす
